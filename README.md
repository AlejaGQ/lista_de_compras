# lista_de_compras
Se realiza una app en Python para realizar una lista de compras. 

#Características
Se crea un bucle while que se ejecute hasta que el usuario decida salir.
El bucle contiene el siguiente menú de opciones:
Agregar artículo
Eliminar artículo
Mostrar lista
Salir
